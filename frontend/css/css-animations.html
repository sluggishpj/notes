# animation-timing-function

动画播放方式，跟 transition-timing-function 取值一样，一个或多个 <easing-function> (opens new window)

/* Keyword values */
animation-timing-function: ease;

animation-timing-function: ease-out;
animation-timing-function: ease-in-out;
animation-timing-function: linear;
animation-timing-function: step-start;
animation-timing-function: step-end;

/* Function values */
animation-timing-function: cubic-bezier(0.1, 0.7, 1, 0.1);
animation-timing-function: steps(4, end);

/* Multiple animations */
animation-timing-function: ease, step-start, cubic-bezier(0.1, 0.7, 1, 0.1);

# animation-direction

动画播放方向

/* Single animation */
animation-direction: normal; /*default 结束后立刻回到原来位置进行下一轮动画 */
animation-direction: reverse;
animation-direction: alternate;
animation-direction: alternate-reverse;

/* Multiple animations */
animation-direction: normal, reverse;
1
2
3
4

  • reverse: 跟正常动画相反顺序
  • alternate: 多次动画时，一次正，一次反，表现就是来回动画，且首次是正常动画顺序
  • alternate-reverse: 跟 alternate 反方向来回动画，首次是反方向动画

# animation-fill-mode

设置 CSS 动画在执行之前和之后如何将样式应用于其目标

/* Single animation */
animation-fill-mode: none;
animation-fill-mode: forwards;
animation-fill-mode: backwards;
animation-fill-mode: both;

/* Multiple animations */
animation-fill-mode: none, backwards;
1
2
3
4
5
6
7
8
  • none: 当动画未执行时，动画将不会将任何样式应用于目标，而是已经赋予给该元素的 CSS 规则来显示该元素
  • forwards: 动画执行之后，目标将保留由执行期间遇到的最后一个关键帧计算值。
  • backwards: 开始应用动画属性时(delay 之前)，目标立即应用 第一个关键帧中定义的值
  • both: 同时应用 forwards 和 backwards 的规则

# animation-name

通过名称指定应用哪些 @keyframes (opens new window) 定义的动画。名称必须和 @keyframes 定义的名称全等(===)才匹配上

#target {
  animation-name: slide;
  /*other*/
}

@keyframes slide {
  from {
    background-color: orange;
    color: #000;
    margin-left: 0;
  }

  to {
    background-color: orange;
    color: #000;
    margin-left: 80%;
  }
}

# 打字动画

核心思路就是让容器的宽度成为动画的主体：把所有文本包裹在这个容器中，然后让它的宽度从 0 开始以步进动画的方式、一个字一个字地扩张到它应有的宽度。

长度单位 ch 指代字符0的宽度，在英文等宽字体中可以使用它，但是中文却不行（不知道为啥） 中文试了下，中文字符的宽度是 font-size大小

# 沿环形路径平移动画

内部图片需要抵消掉外部的变换

@keyframes spin {
  to {
    transform: rotate(1turn);
  }
}

.avatar {
  width: 200px;
  height: 200px;
  border-radius: 50%;
  background: #fec21e;
  text-align: center;
  animation: spin 6s infinite linear;
}

.avatar > img {
  width: 50px;
  height: 50px;
  border-radius: 50%;
  animation: spin 6s infinite linear reverse;
  position: relative;
  top: -25px;
}
