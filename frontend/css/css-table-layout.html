# 单元格边框

# border-collapse

border-collapse: collapse | seperate;
1
  • collapse: 合并单元格边框，相邻的单元格边框中心点重合
  • seperate: 不合并单元格边框，相邻单元格间距由 border-spacing 决定

border-collapse 和 border-spacing 都是设置在 table 元素上
