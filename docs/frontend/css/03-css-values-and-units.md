## 长度单位

- 绝对长度单位（很少使用）:英寸（`in`）、厘米（`cm`）、毫米（`mm`）、点（`pt`）
- 相对长度单位:
  - `em`：相对于父元素`font-size`大小
  - `rem`: 相对根元素(`<html>`)的`font-size`大小
  - `vw`: 1% viewport 的宽度
  - `vh`: 1% viewport 的高度
  - `vmin`: 1% `Math.min(viewport宽，viewport高)`
  - `vmax`: 1% `Math.max(viewport宽，viewport高)`

## 角度单位

- `deg`: 度，比如一个圆是 `360deg`
- `rad`: 弧度，比如一个圆是 `2πrad`
- `turn`: 圈，比如一个圆是 `1turn`
